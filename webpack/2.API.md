# Webpack API

## CLI

### build

`build|bundle|b [entries...] [options]`
默认命令，可省略

### configtest

`configtest|t [config-path]`
校验 webpack 配置

### info

输出有关系统的信息

### init

初始化一个新的 webpack 配置

### loader

生成一个 loader 仓库

### plugin

生成一个 plugin 仓库

### serve 
`serve|s [options]`
运行 webpack-dev-server

### watch
`watch|w [entries...] [options]`
运行 webpack 并监听文件变化
