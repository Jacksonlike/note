## 三种使用 CSS 的方法

### 外部 CSS

外部样式在 HTML 页面 `<head>` 部分内的 `<link>` 元素中进行定义:

```html
<link rel="stylesheet" type="text/css" href="style.css">
```

### 内部 CSS

内部样式是在 head 部分的 `<style>` 元素中进行定义

### 行内 CSS（内联样式）

行内样式可用于为单个元素应用唯一的样式:
```html
<h1 style="color:blue;text-align:center;">This is a heading</h1>
````
